# SSH Allowed Signers

[.ssh-allowed-signers](.ssh-allowed-signers) contains my SSH public keys trusted for signing commits and tags on GitHub.

## Format

```text
principal key-type public-key-data comment
```

## Current Trusted Keys

- **<github.com@idunno.org>** - 1password backed signing key

## Verification

### Commit Signing

To verify tags using these keys:

```bash
# Configure Git to use this file
git config gpg.ssh.allowedSignersFile .ssh-allowed-signers

# Verify a tag
git verify-commit <commit>
```

### Tag Signing

To verify tags using these keys:

```bash
# Configure Git to use this file
git config gpg.ssh.allowedSignersFile .ssh-allowed-signers

# Verify a tag
git verify-tag <tag>
```
